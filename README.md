# Studio DG Apps Website

This repository is intended to be published as the GitHub Pages user site for Dgreen-28.

## Required repository name

Create the repository with this exact name:

```text
Dgreen-28.github.io
```

GitHub Pages should publish the site at:

```text
https://dgreen-28.github.io/
```

AdMob should be able to find app-ads.txt at:

```text
https://dgreen-28.github.io/app-ads.txt
```

## Files

Upload these files directly into the root of the repository:

```text
index.html
privacy.html
support.html
styles.css
app-ads.txt
README.md
```

Do not upload them inside an extra folder.

## App Store Connect

For Say What Now?, set the Marketing URL / Developer Website to:

```text
https://dgreen-28.github.io/
```

Then return to AdMob and click **Check for updates**.
